# Freeway Speed-Flow Relationship Analysis

## Summary

Downloaded and parsed official VD XML.GZ snapshots to reconstruct minute-level speed-flow observations for a selected freeway detector.

## What This Project Shows

- Built a reproducible XML.GZ parsing pipeline.
- Aggregated lane-vehicle records into volume-weighted minute-level speed-flow observations.
- Compared morning and afternoon traffic-flow conditions.
- Tested linear and quadratic speed-flow models.

## Key Results

- Parsed 120 official minute-level snapshots.
- Extracted 720 lane-vehicle records.
- Afternoon mean flow increased by 104.67% relative to the morning period.
- Mean speed declined by 29.64 km/h.
- Detected 33 low-speed afternoon minutes and a 32-minute continuous episode.

## Limitations

This is a selected-detector analysis and should not be interpreted as freeway-wide capacity estimation.
