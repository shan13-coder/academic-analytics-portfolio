# E-Commerce GMV Growth Analysis

## Summary

Diagnosed GMV slowdown for an e-commerce platform using seven relational datasets covering orders, customers, payments, products, reviews, and item-level transactions.

## What This Project Shows

- Built an analytics pipeline across 543,533 source records.
- Created an order-level BI table to avoid duplicated GMV after joins.
- Decomposed GMV decline into order volume, purchasing customers, order frequency, and average order value.
- Identified high-priority customers using recency and monetary thresholds.
- Extended the assignment with PSI, cohort retention, category gap, and delivery-review analysis.

## Key Results

- 2018-08 GMV was 14.43% below the 2017-11 peak.
- Decline was mainly driven by lower orders (-13.68%) and purchasing customers (-13.06%).
- Identified 15,481 high-priority customers.
- Delayed orders had lower average review scores than on-time orders (2.57 vs. 4.30).

## Limitations

Conversion scenarios are assumption-based estimates, not A/B-tested campaign outcomes.
