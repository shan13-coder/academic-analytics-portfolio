# Public Transport Accessibility with QGIS

## Summary

Analysed public-transport accessibility to tourist sites in Hsinchu East District using household, attraction, administrative-boundary, and bus-stop datasets.

## What This Project Shows

- Processed household-level GIS records.
- Deduplicated route-stop records into unique stop locations.
- Built 300 m, 500 m, and 800 m accessibility sensitivity scenarios.
- Extended buffer-only analysis with route-aware accessibility screening.

## Key Results

- Processed 202,470 Hsinchu City household records.
- Analysed 104,728 East District households, 53 villages, and 20 attractions.
- Deduplicated 2,624 route-stop records into 1,165 unique stops.
- 500 m buffer-only accessibility = 48.99%.

## Limitations

Route-aware results do not include schedules, frequency, transfers, travel time, or pedestrian-network distance.
