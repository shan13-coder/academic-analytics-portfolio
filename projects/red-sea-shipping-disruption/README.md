# Red Sea Shipping Disruption Analysis

## Summary

Reconstructed a labelled before/after shipping-network impact dataset from a peer-reviewed study on the Red Sea crisis.

## What This Project Shows

- Converted paper-reported and figure-labelled values into a structured port-route impact dataset.
- Compared October 2023 and February 2024 network indicators.
- Ranked port-level capacity winners and losers.
- Analysed transshipment-port exposure and route-level resilience indicators.

## Key Results

- 27 labelled port-route impacts across 22 ports.
- 25 of 26 analysed services rerouted via the Cape of Good Hope.
- 53 additional vessels and approximately 1.0 million TEU of deployed capacity.
- Weekly port-call decline: Asia-Mediterranean -13.5%, Asia-Europe -8.4%.
- Transshipment ports accounted for 85.24% of gross labelled capacity impacts.

## Limitations

Dataset includes only values explicitly stated or labelled in the paper. It is not a complete AIS or global port-census analysis.
