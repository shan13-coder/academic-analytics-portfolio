# DHL Supply Chain Digital Transformation

## Summary

Analysed DHL's automotive logistics and digital-transformation strategy through a source-KPI matrix and assumption-based distribution-network scenarios.

## What This Project Shows

- Mapped automotive supply-chain challenges to DHL logistics mechanisms and potential KPIs.
- Analysed 3PL and Lead Logistics Partner roles in inbound logistics, line-side delivery, spare-parts distribution, and ERP integration.
- Compared centralized DC, local DC, and three-layer Micro-LDC scenarios.
- Tested service-radius and speed assumptions across nine sensitivity cases.

## Key Results

- Built a 7-row supply-chain challenge to KPI evidence matrix.
- Compared 3 distribution-network designs.
- Tested 9 assumption-based sensitivity scenarios using 40-60 km/h average-speed assumptions.

## Limitations

This project does not use DHL internal operational data. Scenario results should not be described as measured DHL performance improvements.
