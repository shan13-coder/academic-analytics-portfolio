# Data Notes

This repository is a public portfolio version of the project workspace.

## What Is Included

- Project reports and status notes.
- Reproducible Python and JavaScript scripts.
- Selected summary CSV files and JSON headline metrics.
- CV-ready bullet point drafts and limitation notes.

## What Is Excluded

- Raw PDF reports, journal articles, and slide decks.
- Large raw datasets and generated BI-wide tables.
- GIS shapefiles and downloaded archives.
- Taiwan Freeway Bureau minute-level XML.GZ snapshots.
- Google Drive temporary HTML files.
- Generated Excel workbooks and rendered preview images, unless specifically needed.

## Why Raw Data Is Excluded

Several source files are large, copyrighted, classroom-provided, or more appropriately downloaded from official/public sources. The public repository keeps only the minimum evidence needed to make the analysis understandable and auditable.

## Important Claim Boundaries

- Red Sea project: reconstructed only values explicitly labelled or stated in the peer-reviewed paper; it is not a complete AIS or port-census dataset.
- DHL project: scenario model is assumption-based; it is not based on DHL internal operational data.
- Urban mobility project: clustering is descriptive and does not establish causal policy effects.
- E-commerce GMV project: conversion scenarios estimate potential GMV recovery and are not A/B-tested campaign outcomes.
- VD/ETC project: reported agreement compares two data-source classifications and should not be described as ground-truth accuracy.
