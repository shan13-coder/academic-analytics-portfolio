# Academic Projects Portfolio

Target applications: Data Science and Supply Chain Management

This portfolio prioritizes projects with reproducible data pipelines, defensible quantitative evidence, model validation, and clear operational relevance. Results are presented with explicit limitations rather than inflated impact claims.

## Portfolio Principles

- Use the featured projects instead of listing every course assignment.
- Lead with verified scale, method, model metric, and decision relevance.
- State association rather than causation when the design is observational.
- Discuss limitations before an interviewer has to uncover them.
